Why PMOI?

Developed by: Albert Lua, Daniel Trager, Jason Leong, Paul Shin, Sai Coumar

PMOI (Putta Mask On It!) utilizes a web interface to send video feed to a trained AI engine model in Python to analyze if a mask is being worn correctly. If it is not, it will notify there is a violation on mask-wearing.

Created for the "Code for Good 2020" Hackathon by HackDuke in the "Health" category.
